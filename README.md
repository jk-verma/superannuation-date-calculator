# Superannuation Date Calculator

A simple web application to calculate:

- Age as on a selected date
- Date of superannuation based on selected retirement age
- Remaining service duration from the selected date to superannuation

## Features

- Date selectors for:
  - Date of Birth
  - Age As On Date
- Superannuation age options:
  - 60 years (General Central Govt)
  - 62 years (Doctors / Scientists)
  - 65 years (Professors / Special Cadre)
- Calendar picker support using Flatpickr
- Instant calculation in the browser (no backend required)

## How to Use

1. Open `index.html` in a web browser.
2. Select Date of Birth.
3. Select Age As On Date.
4. Choose the applicable superannuation age.
5. Click **Calculate Superannuation**.

The app displays:

- Age as on selected date (Years, Months, Days)
- Superannuation age
- Date of superannuation
- Remaining service (Years, Months, Days)

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- [Flatpickr](https://flatpickr.js.org/) (via CDN)
